This is how I created this image

I wanted to create a more complex image manipulation by merging two pictures together, which was a good challenge for me.

  1. First, I started with the picture of the window as the background and used the pen tool to select the areas of the window that were showing the outside world.
  2. Then I inserted the picture of the woman diving inside this one and used it as a clipping mask to the previous area that I've selected with the pen tool
  3. I, then, with the pen tool again selected the area of the window that was mirroring the outside world and used the picture of the woman diving as a clipping mask again. I also reduced the opacity of this layer, to create an effect that the window is merely mirroring the outside world
  4. From the woman diving picture, I've selected the woman and with the selection created a new layer, containing just the diving woman cut out of the picture
  5. I've positioned the diving woman into the window, so it looks like as if she was jumping out of the window into the water outside.
  6. To make it look more authentic, I've used the soft round brush tool 13 to create a splash around her hands, with a colour that I've sampled from the water with the eyedropper tool

This is how I created this image

I wanted to challenge myself in not only erasing objects and people from the photo but also changing the perspective of the image

  1. I duplicated the picture
  2. Then, I used the transform tool to straighten the image, so the horizontal line of the pool lays on the X-axes.
  3. I used the clone stamp tool to get rid of the shadow of the man first, and then the corners of the image that were created after I've transformed the picture. The reason why I've duplicated the background layer at the beginning, was to keep the same dimensions of the image, so I don't have to crop it drastically.
  4. With each new layer, I've used the clone stamp tool to delete the man, the handrail and correct the waves so they look authentic and not repeat themselves.
  5. Then, I've used the transform tool again only for the part of the image with the pool to create a flat effect.
  6. I've used the clone stamp tool to correct the colors of the pool and the black stripes in the pool.
  7. With the marquee tool, I've selected the top part of the pool and filled it with one colour, which I've sampled with the eyedropper tool.
  8. Finally, I've corrected the texture of the top part of the water in the pool by using the clone stamp tool.
  9. I saved it for web in the highest JPEG quality, not to lose the quality, as it was a photograph

This is how I created this image

I wanted to create an original SVG logo that I could reproduce and manipulate in all of my mockups. I had an idea in mind, but could not find anything on the internet, so I've decided to create one myself

  1. I've drawn a coffee cup shape on a paper, I've taken a picture of it and opened it in Photoshop
  2. Here, I've used the pen curvature tool to select the outline of the drawn coffee mug and filled it with blue colour
  3. As I didn't want the picture to be a full coffee mug but just a cut out, I've masked an area of the layer that I've selected with the pen tool
  4. I've saved it as a PNG-8 with 2 colours, which allowed it to be a really small sized image
  5. I've opened the PNG in Illustrator to be able to be saved as an SVG.
  6. I've used this image as a logo for the company logo in my mockups.

This is how I created this image

I have seen images of photos recreated into cartoons and wanted to try it myself

  1. I wanted to create a cartoon effect.
  2. I've duplicated the background layer, selected the girl in the picture and created a mask from the selection.
  3. Then, from the background layer I've created a solid colour layer of pink. I've also used the brush tool and created a splash effect with just a few mouse click with the brush around the girl.
  4. All the layers apart from the background layer I've selected and converted to a smart object and duplicated it 2 times, so that I get 3 identical layers.
  5. First layer: I went to the filter gallery and I've chosen the poster edge effect and played around with the thickness, intensity and polarization.
  6. Then, I added an unsharp mask from the filter menu, to the image to sharpen it.
  7. Then, I went to the filter menu again and chose the oil paint filter.
  8. I played around with filters, until I've achieved a result that I liked.
  9. Second layer: I applied the same poster edge effect from the filter gallery but with different thickness, intensity and polarization. I've also applied the oil pain to this layer and changed the blending of this layer to overlay and reduced the opacity, so the effects are not too drastic but only subtle.
  10. For the third layer, I've chosen the high pass effect and changed the layer blend mode to overlay to give it a bit more texture
  11. I've also adjusted levels, vibrance and saturation
